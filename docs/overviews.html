文档实用程序¶

用Python编写，用于通用和专用¶

Docutils是一个开源文本处理系统，用于将明文文档处理成有用的格式，例如HTML，LaTeX，手册页，开放文档或XML。 它包括 reStructuredText ，易于阅读，易于使用，你看到的是什么，你得到的纯文本标记语言。

概观¶

要快速启动和运行 ，请参阅 README 文件的 Quick-Start 部分。

所有 文档 都可以从 Docutils Project Documentation Overview 获得。

对于开源项目的开发人员来说，反馈 是一个很好的激励因素，非常受欢迎。我们还依靠反馈来确定要实现的功能。 因此，如果你告诉我们你需要什么，你可能会得到它！请将任何反馈发布到 Docutils-users 邮件列表。

关于Docutils的所有 讨论 都发生在 mailing lists 上。

如果您发现 错误 ，请提交 bug report 或发送电子邮件至 Docutils-users 邮件列表。 补丁 提交应该在 patch tracker (首选)中提交或发送到 Docutils-develop (您将被要求将它们发送到 patch tracker)。

我们欢迎各种 贡献 。有一个 To Do list 充满了等待冠军的有趣想法。如果您对如何扩展 Docutils 有任何疑问，请随时在 Docutils-develop 邮件列表上询问。

运行Docutils的唯一 要求 是 Python

  • 在Docutil 0.14之后，Python 2.4和Python 2.5支持将被删除。

  • Docutils 0.10放弃了Python 2.3支持。

  • 对于Docutils 0.5，需要Python 2.2.1+。

  • 最高为Docutils 0.4，需要Python 2.1。

  • 从版本0.6开始，Docutils与Python 3兼容(有关详细信息，请参阅 README)。

Docutils正在积极开发中，API应被视为 实验 ，可随时更改。虽然 任何时候 是经过认真思考和讨论的时间。

下载¶

注解

sourceforge直接支持快照。整个行李箱是包装的文件，沙箱，perst/perl 和 web:

https://sourceforge.net/p/docutils/code/HEAD/tarball

或者只有 docutils 包:

http://sourceforge.net/p/docutils/code/HEAD/tree/trunk/docutils/

单击下载快照按钮。

下面列出的 snapshosts 是在网站更新中创建的。

我们建议您使用来自 Docutils’ Subversion repository 的 snapshot 。 快照通常包含 更多功能 和 更少的错误 比 “official” 发布—他们不仅仅是开发人员！

最新的 发布包 是 docutils-0.14.tar.gz 。 它于2017年8月3日发布，主要供分销商使用。请参阅 release notes 以获取自上一版本以来的更改列表。

Sandbox 包含实验性的贡献代码。 您可以下载完整的 snapshot of the Sandbox 或 browse the Sandbox 来下载子集。
